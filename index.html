Välkommen till HTML och CSS bootcamp sidan

HTML

HTML står för "Hyper Text Markup Language" och är det språk som alla webbsidor är byggda av. Det berättar hur en hemsida ska se ut och är alltså ett så kallat sidbeskrivningsspråk. Med hjälp av så kallade HTML-taggar kan du till exempel placera in bilder eller tabeller .

CSS
